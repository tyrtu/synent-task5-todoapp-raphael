# Synent Task 5 — To‑Do App

Simple vanilla JavaScript to-do application.

Features:
- Add tasks, mark complete / uncomplete
- Filter: All / Active / Completed
- Delete single task, clear completed
- Persistent storage via `localStorage`

Run:

1. Open `index.html` in your browser, or serve the folder:

```powershell
cd synent-task5-todoapp
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Files:
- `index.html` — UI and markup
- `styles.css` — styles
- `script.js` — application logic (add, toggle, delete, save/load localStorage)

Notes:
- Uses `crypto.randomUUID()` to generate unique IDs for todos.
